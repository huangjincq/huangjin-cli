# huangjin-cli
### 1、基本使用

**step1**：加载全局安装包
```javascript
npm install -g huangjin-cli
```
**step2**：初始化一个项目
```javascript
huangjin init helloworld
```
// helloworld就是项目名

**step3**：写入项目描述和作者信息

**step4**：启动项目

```javascript
cd helloworld

npm install && npm start
```


### 2、命令说明

1、只有安装了 huangjin-cli 的工具包，才可以全局使用 huangjin 命令。

2、huangjin 命令提供了 create 的方式，对项目进行初始化工作。
